 # 05 - Frontend (Vite)
 
 ## Paso 1: Crear proyecto
 
 - `npm create vite@latest`
 - Elegir Vanilla JS o React (segun el enunciado).
 
 ## Paso 2: Pantallas minimas
 
 - Dashboard con metricas.
 - CRUD de empleados.
 - CRUD de entrenamientos.
 - Reportes SQL (10 consultas).
 - Feedback Mongo.
 
 ## Paso 3: Consumo de API
 
 - Crear un modulo `api.js` con funciones fetch.
 - Usar `GET` para listar y `POST` para crear.
 
 ## Paso 4: Validaciones
 
 - Formularios con validaciones basicas.
 - Mensajes de error al usuario.
