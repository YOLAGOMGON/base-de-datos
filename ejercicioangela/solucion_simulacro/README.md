 # Solucion simulacro M4 BD Fullstack
 
 Esta carpeta contiene una guia paso a paso para construir la solucion
 del simulacro. Esta documentado por carpetas y explica las conexiones
 entre SQL, MongoDB, backend y frontend.
 
 ## Estructura
 
 - `backend/`: aqui ira el codigo del servidor Express.
 - `frontend/`: aqui ira la SPA con Vite.
 - `sql/`: scripts SQL (DDL, DML, vistas, consultas).
 - `mongo/`: estructura y ejemplos de colecciones.
 - `docs/`: documentacion paso a paso.
 
 ## Como usar esta guia
 
 1. Lee `docs/01_estructura.md` para ver la estructura completa.
 2. Sigue `docs/02_sql.md` para crear el modelo relacional.
 3. Sigue `docs/03_mongo.md` para el modelo NoSQL.
 4. Sigue `docs/04_backend.md` para levantar el backend.
 5. Sigue `docs/05_frontend.md` para la SPA.
 6. Revisa `docs/06_conexiones.md` para entender todas las conexiones.
 
 ## Nota de seguridad
 
 Nunca subas credenciales reales a git. Usa variables de entorno.
