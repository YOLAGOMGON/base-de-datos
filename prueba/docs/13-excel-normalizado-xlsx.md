13 - Excel normalizado con multiples hojas

Objetivo
Generar un Excel con hojas separadas por entidad (normalizado).

Comando
npm run excel:normalized

Archivo generado
- data/ventas_normalizado.xlsx

Hojas incluidas
- clientes
- productos
- ventas
- detalle_venta

Uso en la prueba
- Sirve como evidencia de normalizacion.
- Puedes exportar cada hoja a CSV para cargar en Postgres.
